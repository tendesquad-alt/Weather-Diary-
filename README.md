# Weather-Diary-
**Автор:** [Воронин Лев]
**Дата** 30.04.2026
## Описание

Приложение для ведения дневника погоды с графическим интерфейсом. Позволяет:
* Добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков.
* Фильтровать записи по дате и температуре.
* Сохранять и загружать данные в формате JSON.


## Как запустить приложение

**Требования:**
* Python 3.6 или выше;
* библиотека Tkinter (обычно идёт в комплекте с Python).

**Запуск:**
1. Откройте терминал или командную строку.
2. Перейдите в директорию проекта: `cd path/to/WeatherDiary`.
3. Запустите приложение: `python main.py`.

## Примеры использования

**Тест 1: Добавление записи о погоде**
1. Заполните поля:
   * **Дата:** 2024-01-15;
   * **Температура:** 5.5;
   * **Описание:** Солнечно, лёгкий ветер;
   * **Осадки:** Нет.
2. Нажмите **«Добавить запись»**.
3. Убедитесь, что запись появилась в таблице.

## Особенности реализации

* **Валидация данных:** программа проверяет заполнение всех полей, формат даты и корректность температуры.
* **Сохранение между запусками:** все записи сохраняются в `weather_records.json`.
* **Гибкая фильтрация:** можно фильтровать одновременно по дате и минимальной температуре.
* **Интуитивный интерфейс:** все элементы управления расположены логично и понятно.
